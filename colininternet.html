What is the internet?

The internet is referring to a series of networks between different devices where the users get connected in order to access the desired websites, such as this one you're reading.

Foundations of the internet (1969-1980)

The first traces of the internet date back to 1969 when they were only available in large factories and power plants to communicate with other entities working for a common job. These computers back then were very big compared to the ones we use today.

The Start of a New Era (1980-present)

The revolutionary age for the internet and computers wouldn't come until the 1980s when personal computers were developed, too. Because of this, thousands of people since then were able to connect personally without needing for a job in a large industry, causing a system of thousands of network connections.

Internet Anatomy

Websites such as the one you're viewing at the moment used to be commonly created using HTML and CSS. Nowadays, many websites are made using other software, but the first two mentioned should give you an idea how web pages are saved as a file with the text inputs.

Subpages of a website are saved in a zipped folder, where the files needed for the website need to be unzipped in order to properly set it up. Note that the subpages of the website would need to be located inside subfolders too in order for them to work on the website.

Once you connect to a network, you will attain access and be free to access any website you will, or in this case, of the sample website above, click on the said website on the search bar, and you will be able to retrieve the information you want.
